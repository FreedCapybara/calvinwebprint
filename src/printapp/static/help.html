• Calvin Web Print

Help & Support

If your problem is not addressed on this page or if you have any suggestions, send an email to calvinwebprint@gmail.com.

Login

To login, type in your Calvin username (not email) and passphrase (the same passphrase used to login on Portal and Moodle) and click the “Sign in” button.

Setup

Note: You only need to setup your account once.

Connect your account

In order to print, you must connect your account to Google Cloud Print. Connect to Google Cloud Print by simply clicking the “Connect your account” button. Make sure you are logged in to your StudentMail account when you are allowing Calvin Web Print permission to manage your printers.

Add Calvin Printer

If you have never used uniFLOW to print, you will need to add Calvin's uniFLOW printer to your Google Cloud Print account. To do this, click “Add Calvin Printer” and a new window will pop up. Click “Accept” then close the window. Now you are all set to print using Calvin Web Print.

Print

First select a file that you would like to print. The file must be a Word document, PDF, OpenOffice document (.odt), or picture file format. You may click “Cancel upload” at any point to cancel the upload. Specify the number of copies you would like, as well as any of the different options. Once the file is uploaded, click “Submit.” Now your file is ready to be printed at a printer.

If you are wondering why you cannot print your document double-sided and stapled, we disallow it is because the pages of double-sided documents get stapled out of order.

If you are printing more than one copy and want to know what the “Collate” option means, refer to the diagrams on this page.

Print Queue

Your Print Queue shows all the files that are waiting to be released at a printer. Find a printer on campus and swipe your ID card to print these files. Files are automatically removed from your queue after 24 hours.

Printer Locations

Note: If you are on a mobile device, the campus map will not appear.

Use the drop down menu to click on the building that you would like to look up. If you are using a computer, you may also click on or hover over the different buildings on the map to find a desired printer.
